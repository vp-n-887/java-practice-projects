# 🎉 New Year Countdown

A simple Java console application that performs a countdown based on user input and displays a New Year greeting after the countdown ends.

## Features

- User-defined countdown timer
- One-second delay between each number using `Thread.sleep()`
- Console-based countdown animation
- Displays a celebratory message at the end

## Technologies Used

- Java
- Scanner Class
- For Loop
- Thread.sleep()

## How to Run

Compile the program

```bash
javac NewYearCountdown.java
```

Run the program

```bash
java NewYearCountdown
```

## Sample Output

```
Countdown: 5

5
4
3
2
1

🎉 Happy New Year! 🎉
```
