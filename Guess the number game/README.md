# 🎯 Number Guessing Game

A simple Java console game where the computer randomly generates a number, and the player keeps guessing until the correct number is found.

## Features

- Random number generation
- Unlimited guessing attempts
- Tracks the number of attempts
- Simple console-based gameplay

## Technologies Used

- Java
- Scanner Class
- Random Class
- Do-While Loop

## How to Run

Compile the program

```bash
javac NumberGuessingGame.java
```

Run the program

```bash
java NumberGuessingGame
```
