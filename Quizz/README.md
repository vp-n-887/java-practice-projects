# 🧠 Quiz Game

A Java console-based quiz game where players answer multiple-choice questions and receive a score based on their performance.

## Features

- Multiple-choice questions
- Score tracking
- Instant feedback for each answer
- Performance message based on final score
- Console-based gameplay

## Technologies Used

- Java
- Scanner Class
- One-Dimensional Arrays
- Two-Dimensional Arrays
- Enhanced For Loop
- Switch Statements

## How to Run

Compile the program

```bash
javac QuizGame.java
```

Run the program

```bash
java QuizGame
```

## Sample Gameplay

```
----------- QUIZ GAME -----------

Question: 2 + 3

1. 4
2. 5
3. 6
4. 7

Enter your answer:
2

Correct!
```
