# ⚖️ Weight Converter

A simple Java console application that converts weights between kilograms and pounds.

## Features

- Convert Kilograms to Pounds
- Convert Pounds to Kilograms
- Interactive console menu
- Simple and easy-to-use interface

## Technologies Used

- Java
- Scanner Class
- Conditional Statements

## How to Run

1. Compile the program

```bash
javac WeightConverter.java
```

2. Run the program

```bash
java WeightConverter
```

## Sample Output

```
Weight Converter

Enter option
1. Kilograms to Pounds
2. Pounds to Kilograms

Enter the weight:
10

The weight in pounds is 22.046
```
