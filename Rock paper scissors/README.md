# ✊✋✌️ Rock Paper Scissors

A Java console-based Rock Paper Scissors game where the player competes against the computer. The computer randomly selects a move, and scores are updated based on the game outcome.

## Features

- Play against the computer
- Random computer moves
- Score tracking
- Play multiple rounds
- Console-based interaction

## Technologies Used

- Java
- Scanner Class
- Random Class
- Arrays
- Switch Statements
- Loops

## How to Run

Compile the program

```bash
javac RockPaperScissors.java
```

Run the program

```bash
java RockPaperScissors
```

## Sample Gameplay

```
------------ ROCK PAPER SCISSORS ------------

Player Score: 5
Computer Score: 5

Computer move:
paper

Enter your move:
rock

Do you want to play again?
1 - Yes
0 - No
```
