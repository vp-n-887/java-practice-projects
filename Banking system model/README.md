# 🏦 Banking System

A Java console-based banking system that allows users to create a bank account, deposit money, withdraw money, and view account details through a menu-driven interface.

## Features

- Create a bank account
- View account details
- Deposit money
- Withdraw money
- Balance validation during withdrawal
- Menu-driven console interface

## Technologies Used

- Java
- Scanner Class
- Methods
- Switch Statements
- Loops

## How to Run

Compile the program

```bash
javac BankingSystem.java
```

Run the program

```bash
java BankingSystem
```

## Sample Menu

```
========== Banking System ==========

1. Create Account
2. View Account Details
3. Withdraw Money
4. Deposit Money
5. Exit

Enter your choice:
```
